诡异怪谈

游戏概述

欢迎进入这个充满诡异的世界。每个故事背后都隐藏着难以置信的真相，而你的任务就是通过智慧和勇气，揭开这些诡异事件的面纱。但要记住，有些真相可能会让你后悔知道...

生存法则

问答规则

  • 每个故事仅有20次提问机会，用尽即为失败
  • 只能问是/否问题，任何其他形式都将被诅咒
  • 系统会分析你的问题，但答案可能会令你不安

提示系统

  • 提示可能会帮助你，但每次使用都要付出代价
  • 使用提示将失去5点灵魂点数
  • 每次使用后需等待30秒的冷却期
  • 提示会随着你的深入逐步解锁，但要谨慎使用

灵魂点数

  • 初始获得100点灵魂点数
  • 每次求助都将损失一部分灵魂
  • 最终评分 = 剩余灵魂 × 真相接近度

深度等级

  • 第一层 ★★☆☆☆ 入门层
  • 第二层 ★★★☆☆ 深层
  • 第三层 ★★★★☆ 深渊

生存技巧

  • 细心观察每个细节，它们都可能是关键线索
  • 思考人物行为背后的隐藏动机
  • 注意时间和环境的反常之处
  • 将零散的线索串联成完整的真相
  • 有时真相就藏在最不经意的角落

警告事项

1. 每个故事都经过精心编织，真相可能令人不安

2. 表象下的真相往往出人意料，但必定符合某种规则

3. 提示系统会指引你靠近真相，但代价不菲

4. 必须通过前一层的考验才能进入更深的层次
